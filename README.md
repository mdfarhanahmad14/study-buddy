# 📚 AI-Powered Study Buddy

An AI-powered web application built with Streamlit and Google Gemini API 
that helps students learn faster and smarter.

## 🚀 Features

- *Topic Explainer* — Enter any topic and get a simple, clear explanation
- *Notes Summarizer* — Paste your notes and get a concise summary
- *Quiz Generator* — Generate MCQ quizzes on any topic instantly

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## ⚙️ Installation

1. Clone the repository
2. Install dependencies
3. Create a .env file and add your Gemini API key
4. Run the app
